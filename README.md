# portfolio-mysql
Portfolio of reports developed in MySQL, focused on corporate environments and ERP systems. The projects demonstrate SQL queries for data extraction, consolidation and analysis, supporting administrative, financial and operational decision-making using SQL best practices.

## Technologies
- MySQL
- SQL

## Topics Covered
- Inventory and stock analysis
- Financial and administrative reports
- Data aggregation and consolidation

## Repository Structure
Each folder contains SQL reports with comments and images explaining the business logic and query purpose.
